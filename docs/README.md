## One-line thesis

> Most apps either **organize tasks**, **plan the day**, or **track time** — almost none do all three _and_ work fully offline. We own that intersection at a mid-tier price.

## Core differentiators

1. **Offline-first** — works with zero connection; local data is the source of truth.
2. **Native time tracking** — plan _and_ actuals in one place (the wedge).
3. **Multi-view** — list · table · calendar, across day & week.
4. **Privacy by default** — local-first; sync is an optional, paid, encrypted layer.
